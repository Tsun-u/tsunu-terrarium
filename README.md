# 🌱 像素生態瓶（Pixel Terrarium）

一個幾乎沒有文字的點陣風療癒放置遊戲：像素小動物在生態瓶裡自己生活、相遇、組成家庭，生下繼承形狀與顏色基因的後代。你只需要溫柔地看著——偶爾摸摸牠們、放顆果實，收集愛心。

**🎮 線上玩：https://tsun-u.github.io/tsunu-terrarium/**

## 這個世界怎麼運作

- **8 隻祖代**從四種形狀（圓、三角、方、菱）和色相環出發
- 形狀是 **superformula 參數基因**：後代＝父母參數插值＋噪聲，圓×三角會生出圓潤的中間形；極低機率突變出星形、齒輪形等全新輪廓
- 顏色是 **HSL 基因**：色相偏向父母其一，極低機率跳色或出現微微發光的體質
- 生命週期：蛋 → 幼年（小圓點，六小時後**形狀揭曉**）→ 成年 → 老年 → 溫柔地飄向夜空**化作星星**——點夜空的星星可以看到是哪個孩子
- 壽命 4～7 天（真實時間）。離線期間世界照常運轉（最多折算 72 小時），回來時會看到「你不在的時候…」圖形摘要

## 互動

| 操作 | 效果 |
|---|---|
| 點小動物 | 資訊小卡（肖像、名字、階段、父母）|
| 卡片開著再點一下 | 摸摸，冒愛心 ❤ |
| 點名字 | 改名（全遊戲唯一的文字輸入）|
| 點空地 | 放果實，附近的孩子會小跑步過來 |
| 🌳 | 族譜：世代樹＋親子連線 |
| ⭐ | 星空畫廊：所有化星的孩子 |
| 🛍️ | 愛心商店：🍎延壽（同一隻越買越貴）、🧵月老紅線（幫兩隻單身孩子牽線）、花叢／石燈（夜裡會亮）／鞦韆／小橋／池塘擴建 |

世界還會不定時發生小驚喜：蝴蝶群、流星、彩虹、螢火蟲、送花、疊疊午睡、花瓣雨——在場看到就有 ❤。

## 技術

純前端零建置：Canvas 2D 點陣渲染、邏輯層（基因／模擬）與表現層（渲染／介面）以 `contract.js` 為唯一介面，邏輯層可直接 `node --test tests/` 驗證。存檔在 localStorage。

開發時可用 `?fast=N` 加速時間（例如 `?fast=3600` 一秒＝一小時），方便觀察完整生命週期。
