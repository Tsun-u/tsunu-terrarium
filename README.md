# 🌱 像素生態瓶（Pixel Terrarium）

一個幾乎沒有文字的點陣風療癒放置遊戲：像素小動物在生態瓶裡自己生活、相遇、組成家庭，生下繼承形狀與顏色基因的後代。你只需要溫柔地看著——偶爾摸摸牠們、放顆果實，收集愛心。

**🎮 線上玩：https://tsun-u.github.io/tsunu-terrarium/**

*[English guide →](README.en.md)*

## 第一次打開：開瓶儀式

新瓶子會先進入開瓶儀式：

1. 幫瓶子取個名字（可以直接用預設的）
2. 決定 8 隻祖代的長相——每格可以自己選形狀跟顏色，或按 🎲 交給命運（🎲🎲 一次全骰）
3. 按 🌱，世界誕生

## 這個世界怎麼運作

- **8 隻祖代**從四種形狀（圓、三角、方、菱）和色相環出發
- 形狀是 **superformula 參數基因**：後代＝父母參數插值＋噪聲，圓×三角會生出圓潤的中間形；極低機率突變出星形、齒輪形等全新輪廓
- 顏色是 **HSL 基因**：色相偏向父母其一，極低機率跳色或出現微微發光的體質
- 生命週期：蛋（會微微搖晃）→ 幼年（小圓點，六小時後**形狀揭曉**）→ 成年 → 老年（走得慢，偶爾停下來望著天空）→ 溫柔地飄向夜空**化作星星**——點夜空的星星可以看到是哪個孩子
- 兩隻單身成年孩子常常相遇，就可能**喜結連理**：紅線亮起、愛心噴泉、鈴聲響起。見面越多次，越容易看對眼
- 成家後的伴侶會走在一起、一起發呆；幼年的孩子會跟在爸媽附近
- 若伴侶先一步化星，還沒老的那位有機會**再遇見新的緣分**；已入老年則會守著回憶終老
- 擺了遊具他們會自己去玩：幼體盪鞦韆、爬上溜滑梯咻一聲滑下來、兩隻一起蹺蹺板、白天在池塘邊潑水、走過小橋散步、夜裡還有皮小孩偷按石燈開關
- 壽命 4～7 天（真實時間）。離線期間世界照常運轉（最多折算 72 小時），回來時會看到「你不在的時候…」圖形摘要

## 互動

| 操作 | 效果 |
|---|---|
| 點小動物 | 資訊小卡（肖像、名字、階段、伴侶、父母）|
| 卡片開著再點一下 | 摸摸，冒愛心 ❤ |
| 點名字旁的 ✏️ | 改名（全遊戲唯一的文字輸入）|
| 點空地 | 放果實，附近的孩子會像皮克敏一樣衝刺過來（一次最多放「在場孩子數」顆）|
| 點夜空的星星 | 看看這顆星是哪個孩子 |
| 點裝飾 | 撿起來搬家（免費）；🗑️ 收起來，之後 ❤0 隨時買回 |

## 技術

純前端零建置：Canvas 2D 點陣渲染、邏輯層（基因／模擬）與表現層（渲染／介面）以 `contract.js` 為唯一介面。邏輯層測試：

```bash
node --test tests/sim.test.js tests/genetics.test.js
```

開發時可用 `?fast=N` 加速時間（例如 `?fast=3600` 一秒＝一小時），方便觀察完整生命週期。
